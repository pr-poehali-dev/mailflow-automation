## Как опубликовать

### VC.ru
1. Регистрация на [vc.ru](https://vc.ru) → личный аккаунт компании MAIL-KA.
2. Раздел «Маркетинг» или «Сервисы».
3. Заголовок не менять (он SEO-оптимизирован).
4. Обложка — сгенерируйте картинку «маркетолог за ноутбуком, российский флаг, email» в Midjourney/Шедевруме.
5. В первый абзац ставим ссылку на mail-ka.ru — Яндекс ценит ссылки в верхней части текста.

### Habr
1. Регистрация на [habr.com](https://habr.com) → песочница (Sandbox).
2. После 1-2 публикаций откроется полноценный аккаунт.
3. Хабы: «Разработка веб-сайтов», «Облачные сервисы», «Python», «PostgreSQL».
4. Тег «Tutorial» поставить нельзя без полноценного аккаунта — выбираем «Разработка».
5. Код-блоки оформить с подсветкой синтаксиса (```python).
